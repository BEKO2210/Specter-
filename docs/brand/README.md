# Specter — Markenmaterial

Logo, Bildmarke und Farben von Specter. Neben den bestehenden PNGs gibt es das
Logo jetzt auch **vektorisiert (SVG)** — verlustfrei skalierbar für Print, große
Formate, Web und Favicons.

## Farben
| Rolle | Hex |
|---|---|
| Navy (Hintergrund / Wortmarke hell) | `#0A1420` / `#133149` |
| Teal (Akzent) | `#14B8A6` (Marke: `#49AEAF`) |
| Weiß (Wortmarke auf dunkel) | `#FFFFFF` |

## Vektor-Logo (SVG)
| Datei | Beschreibung | Untergrund |
|---|---|---|
| [`specter-mark.svg`](specter-mark.svg) | Bildmarke „S"-Hexagon, **Teal + Navy** | hell |
| [`specter-mark-white.svg`](specter-mark-white.svg) | Bildmarke, **Teal + Weiß** | dunkel |
| [`specter-logo.svg`](specter-logo.svg) | Wortmarke-Lockup, einfarbig **Navy** | hell |
| [`specter-logo-white.svg`](specter-logo-white.svg) | Wortmarke-Lockup, einfarbig **Weiß** | dunkel |

Die SVGs wurden aus den Original-PNGs vektorisiert (flaches Design). Die
3D-Faltung der Original-Marke bleibt als PNG erhalten; die SVG-Marke ist die
flache, skalierbare Variante.
