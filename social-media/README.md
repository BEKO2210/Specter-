# Social-Media-Werbematerial

Markengetreue Werbemotive für Specter (Instagram, Facebook & Co.) — dunkles
Navy (#0A1420) mit Teal-Akzent (#14B8A6), deutscher Text, defensive Botschaft.
Erzeugt mit Higgsfield (Nano Banana Pro / Kling 3.0 Turbo).

| [`story-9x16.png`](story-9x16.png) | 9:16 · 2K | **Story / Reel-Cover** | Teal-Schild wehrt rote Angriffe über einem Gewerbegebiet ab — *„Der TÜV für Ihre IT"* |

| [`reel-9x16.mp4`](reel-9x16.mp4) | 9:16 · 1080p · 5 s | **Reel / Story-Video** | Schild pulsiert, Angriffslinien zerstäuben, sanfter Kamera-Push |

## Nutzung

- **Instagram Feed:** `hero-feed-4x5.png` (4:5 nutzt die maximale Feed-Höhe).
- **Story / Reel:** `story-9x16.png` als Standbild oder `reel-9x16.mp4` als Video
  (beide 9:16, randlos). Das Video eignet sich als Loop.
- **Karussell / erklärendes Post:** `attack-path-1x1.png` — visualisiert die
  Kernbotschaft (Angriffspfad) in einem Bild.
- **Facebook:** Feed-Motiv 4:5 oder 1:1; für Stories das 9:16-Material.

Alle Texte sind fest ins Bild gerendert. Für abweichende Claims oder weitere
Formate (z. B. 16:9 für Facebook-Querformat, LinkedIn) einfach neue Varianten
erzeugen lassen.
