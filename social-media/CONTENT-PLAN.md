# Redaktionsplan — Launch-Woche (Woche 1)

Fertiger Social-Media-Plan für **Specter** über die erste Woche: jeder Post,
jedes Format, jeder Text und jeder Hashtag-Satz — abgestimmt auf Instagram,
Facebook und LinkedIn. Alle Motive liegen als fertige Dateien in diesem Ordner
und tragen das **Original-Specter-Logo** (siehe [`README.md`](README.md)).

> **Platzhalter vor dem Posten ersetzen:**
> `[LINK]` = Ziel-URL (Website / Landingpage / Kontaktformular) ·
> `[@HANDLE]` = eure Social-Handles ·
> `[TELEFON/E-MAIL]` = Kontaktweg fürs Erstgespräch.
> Die Texte sind so geschrieben, dass sie ohne erfundene Domain funktionieren.

## 1. Strategie auf einen Blick

| | |
|---|---|
| **Marke** | Specter — Sicherheits-Audit für den deutschen Mittelstand |
| **Kernbotschaft** | „Wir gehen den Weg des Angreifers zuerst — legal, kontrolliert, dokumentiert." |
| **Positionierung** | Der **unabhängige Sicherheits-Check für Ihre IT**: nachvollziehbar, verständlich für Geschäftsführung *und* IT |
| **Zielgruppe** | Geschäftsführung, IT-Leitung & IT-Verantwortliche in KMU/Mittelstand (DACH) |
| **Tonalität** | Seriös, ruhig, kompetent — kein Angst-Marketing, sondern Klarheit & Kontrolle |
| **Sprache** | Deutsch (Sie-Form) |
| **Kanäle** | Instagram (Feed, Story, Reel) · Facebook (Feed, Story) · LinkedIn (Feed) |
| **Frequenz Woche 1** | 1 Hauptpost/Tag + begleitende Stories |

### Content-Säulen
1. **Aufklären** — wie Angriffe wirklich ablaufen (Karussell, Reels).
2. **Beweisen** — belegter Angriffspfad, nachvollziehbare Ergebnisse (Infografik, Benchmark).
3. **Vertrauen** — Unabhängigkeit, Datenschutz, Prüf-Prinzip, deutsche Wertarbeit.
4. **Handeln** — klarer CTA: Sicherheits-Audit / Erstgespräch anfragen.

### Beste Posting-Zeiten (DACH, B2B)
- **LinkedIn:** Di–Do, 08:00–10:00 & 17:00–18:00
- **Instagram:** Mo–Fr, 11:00–13:00 & 19:00–21:00
- **Facebook:** Mo–Fr, 12:00–14:00

### KPIs Woche 1
Reichweite & Impressionen · Profilaufrufe · Interaktionsrate (Saves/Kommentare
> Likes werten) · Link-Klicks · **Anfragen fürs Erstgespräch** (das eigentliche Ziel).

### 🗓️ Tag 3 · Mittwoch — Karussell „So läuft ein Angriff" (Kern-Post der Woche)
**Kanäle:** Instagram Karussell · Facebook Karussell
**Assets:** `carousel-1-aufklaerung.png` → `carousel-5-specter.png` (5 Folien, in Reihenfolge)

**Caption:**
> **So läuft ein Angriff wirklich ab — in 5 Schritten.** ➡️ (swipen)
>
> 1️⃣ **Aufklärung** — der Angreifer sammelt offene Infos: Domains, Mitarbeiter, Technik. Ganz ohne Alarm.
> 2️⃣ **Erstzugang** — eine schwache Stelle genügt: veraltete Software, schwaches Passwort, Phishing-Mail.
> 3️⃣ **Ausbreitung** — von System zu System, bis zu den wichtigen Daten.
> 4️⃣ **Datenabfluss** — Kundendaten, Verträge, Know-how werden kopiert. Oft wochenlang unbemerkt.
> 5️⃣ **Specter geht diesen Weg zuerst** — und schließt die Lücken, bevor ein echter Angreifer sie nutzt.
>
> Kein Angriff beginnt mit einem Knall. Er beginnt leise. Deshalb prüfen wir
> leise mit — und machen ihn sichtbar.
>
